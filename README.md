# TinyCI

![](diagram.png)

TinyCi Main
-----------

Relay board controller by a `W32-ETH01` module.

Input: 5V to 12V

![](tinyci-main.png)

TinyCi USB
-----------

Secondary Relay board controller by the `W32-ETH01` on the main board.

Input: 5V

![](tinyci-usb.png)
